# Redux Toolkit Todo App

A clean and simple **Todo App** built with **React**, **Redux Toolkit**, **React Redux**, and **Tailwind CSS**.  
This project shows how to manage global state in a modern React app using Redux Toolkit, including creating slices, dispatching actions, and accessing state with Redux hooks.

---

## 🚀 Features

- Add new todos
- Update existing todos
- Delete todos
- Manage todo data in global state
- Use `useSelector` to read state
- Use `useDispatch` to trigger actions
- Simple and responsive UI with Tailwind CSS.

---

## 🛠️ Tech Stack

- **React**
- **Redux Toolkit**
- **React Redux**
- **Tailwind CSS**
- **Vite**

## ⚙️ Installation & Setup

Follow these steps to run the project on your local machine:

### 1. Clone the repository

```
git clone https://github.com/MuhammadWaris-55/Redux-Toolkit.git
```

### 2. Install dependencies

```
npm i
```

### 3. Start the development server

```
npm run dev
```
